# 위버스 커뮤니티 포스트 등록, 수정, 삭제 자동화 과제

## 1. 실행 방법
1. 필수 라이브러리 설치: `pip install pytest pytest-playwright python-dotenv`
2. 루트 경로에 `.env` 파일 생성 후 데이터 입력 (TEST_ID, TEST_PW)
3. 실행 명령어: `pytest test_weverse_community.py --headed -s`
